## Atom常用命令:
```
项目内查找文件:ctrl+p
清除一行:ctrl+u
选中一行:ctrl+l
输入命令行:ctrl+shift+p
```
## git 的使用
```
1.添加到暂存区
2.添加提交信息
3. push到远程库

最重要的是你要保存文件(ctrl+s),在右侧的Git窗口下,将待提交的文件移到暂存区中,然后commit,最终push.

需要输入账户和密码.
```
